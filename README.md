# regie-collecteur

Collecteur mail de La Régie (hébergement 24/7 via GitHub Actions).
Relève des boîtes IMAP, trie les messages (Claude), range dans Supabase.

Aucune donnée sensible ici : tout (mots de passe des boîtes, clés Supabase /
Anthropic) vit dans les **secrets GitHub chiffrés**, jamais dans le code.
